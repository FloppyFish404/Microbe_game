Description:
A 2D game made using godot, where a microbe follows the mouse to roam and explore. Gain nutrients and grow by gathering nutrients or killing other hostile microbes with the same mission. Upgrade and evolve your microbes attributes using xp points gained from levelling up. Press space to pause and access the upgrade menu. Hold left click to boost (if upgraded). 

How to play: https://floppyfish404.itch.io/microbe-wars
